자바스크립트의 언어특성
  1. 브라우저에서 실행(run)이 되는 언어(html문서와 같이 작성)
  2. 문법에 맞게 내용을 작성(=source 파일)
  3. 컴파일(compile) => 소스파일(사람이 작성한 것) => 컴파일(=컴퓨터가 알아듣는 말로 변환(2진수)) => 메모리(RAM)에 로딩(컴퓨터가 실행(화면에 결과를 보여줌))
    컴파일(compile)이 없다 => 소스파일(사람이 작성한 것) => 컴파일(=컴퓨터가 알아듣는 말로 변환(2진수))(X) => 문법 체크를 안해서 에러잡기가 어려움 => 메모리(RAM)에 로딩(컴퓨터가 실행(화면에 결과를 보여줌))
  4. 자바스크립트는 컴파일과정이 없고 실행 과정만 있다.
  5. 함수형언어로 구성(함수중요-작성하는 방법 꼭 알아두기, 그냥 외워버리기) (자바=>메서드(method))
  6. 플랫폼(=운영체제)에 상관없이 다 실행이 된다.
  7. 따로 파일을 만들어서 불러올 수가 있다.(소스파일)
